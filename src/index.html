                  function createStar({size, color, x, y}) {
                    const starShape = document.createElementNS(

                    star.setAttribute('transform', 'translate(' + x + ' ' + y + ')';

                    mySVG.appendChild(star);

âœ¨âœ¨Star Parallaxâœ¨âœ¨

ðŸŽ­ Idea: Scrolling Parallax Effect

We have a new star fact and the idea is to use an effect many of you already know.

Classic Parallax Effect

  • Move objects depending on the scroll position and depth
  • translation = maxTranslation * scrollPosition / depth
  • Each star receives a depth bewteen 0 - 5
                  
                    function loop(){
                      // value between 0 and 1 depending on the scroll position
                      parallaxRatio = scrollTop/(totalHeight - viewportHeight);
                      svg.style.setProperty('--ratio', parallaxRatio);
                      requestAnimationFrame(loop);
                    }
                    loop();

                  
                  
                      .bar1 { --depth: 1; }
                      .bar2 { --depth: 2; }
                      .bar3 { --depth: 3; }
                      .bar4 { --depth: 4; }
                      .bar5 { --depth: 5; }
                  
                      .parallax-effect {
                        /** movement = -1 * ratio * maxTranslation */
                        --translationY: calc(-1px * var(--ratio) * var(--maxTranslation));
                        /** actualMovement = movement/depth */
                        transform: translate(0, calc(var(--translationY) / var(--depth) ));
                      }

The body is set to a large height so we can scroll.

The SVG is fixed so we can see it.

Whenever I scroll the ratio, a value between of 0 and 1 is calculated and written into the CSS variables

The CSS Variable is set on the svg but inherited to all stars

The star itself is given a depth value

And with all information it can calculate it's position. The higher the depth the smaller the displacement
